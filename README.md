# 🧩 拼豆工坊（Pindou Workshop）

一个纯前端的拼豆网站：把上传的图片自动转换成拼豆图纸，统计每种颜色的用量，支持在线编辑，并内置现成图纸库。无需后端、无需安装、无需联网，所有处理都在浏览器本地完成。色号采用 **MARD 真实色卡**（A01、F03、ZG3… 字母 + 数字）。

## ✨ 功能

- **图片转图纸**：上传图片，自动分步降采样并把每个格子匹配到最接近的拼豆颜色（CIE-Lab 距离匹配），细节保留好、不易“糊成一团”。
- **参数可调**
  - 横向豆数：滑块 8–200，或数字框直接输入（最高 400）
  - 快速档位：小 48 / 中 80 / 大 120 / 超清 160
  - 调色板：MARD 标准 A–M（221 色）/ MARD 全色卡（291 色）/ 新手精选（72 色）
  - 画质增强：对比度、饱和度、锐化
  - 去除浅色背景、扩散抖动、保持原图比例
- **用量统计**：实时显示总豆数、颜色数、图纸尺寸，并按用量列出每种颜色的**色号 + 数量**（带占比条），点击即选色。
- **在线编辑**：🖌️ 画笔 / 🧽 橡皮 / 💧 取色 / 🪣 填充，支持撤销重做（`Ctrl+Z` / `Ctrl+Y`）。
- **图纸库**：内置 **32 款图纸**，分经典 / 表情 / 动物 / 食物 / 植物 / 节日 / 游戏七类，可分类筛选；一键载入并自动吸附到 MARD 色号。
- **导出 / 保存**：导出图纸 PNG（勾选「色号」时每格标注 MARD 色号）、用量清单 CSV（色号 / 色值 / 数量）、保存/载入项目 JSON（含浏览器本地备份）。
- **显示选项**：网格线、色号（太小会自动放大）、坐标尺、缩放与「适应窗口」。

## 🚀 运行方式

**直接双击 `index.html` 即可在浏览器中打开使用**，无需安装、无需服务器、无需联网。
（全部逻辑打包在 `src/app.js` 这个普通脚本里，不依赖 ES 模块，所以 `file://` 下也能正常运行。）

如果更喜欢用本地服务器访问，可双击 `start.bat`，或在本目录执行：

```bash
python -m http.server 5173   # 然后访问 http://localhost:5173
```

## 📖 快速上手

1. 进入「制作图纸」页，点击或拖拽上传一张图片。
2. 选一个快速档位（或拖动「横向豆数」），需要时调对比度/饱和度。
3. 点击「生成拼豆图纸」，右侧自动统计各颜色用量。
4. 用左下角工具在画布上点击/拖拽进行修改。
5. 在「导出 / 保存」导出 PNG 图纸或 CSV 用量清单。

> 复杂图片想更清晰：横向豆数调到 120–160，开启扩散抖动，并适当增加对比度/饱和度；边缘发虚再勾选锐化。

## 📁 目录结构

```
拼豆/
├─ index.html         页面结构（制作图纸 / 图纸库 / 使用说明 三个页签）
├─ start.bat          Windows 一键启动本地服务器（可选）
├─ README.md
└─ src/
   ├─ style.css       样式
   └─ app.js          全部逻辑（单文件、无构建）
```

`app.js` 内部分为 6 段：① MARD 色卡与 Lab 颜色匹配　② 图片 → 网格（分步降采样 / 对比度 / 饱和度 / 锐化 / 抖动 / 背景去除）　③ 绘图辅助　④ 图纸编辑器（渲染 / 编辑 / 撤销 / 统计 / 导出）　⑤ 内置图纸库　⑥ UI 编排。

## 🎨 关于拼豆色号

- 内置色卡采用 **MARD 拼豆色号体系**，共 **291 色**。色号仅为**字母 + 数字**（如 `A01`、`F03`、`ZG3`），按色系字母分组：
  - **A–H、M 系 = 标准 221 色套装**：A 黄橙系、B 绿系、C 蓝青系、D 蓝紫系、E 粉红系、F 红系、G 灰白黑系、H 紫系、M 灰系等。
  - **P / Q / R / T / Y / ZG = 扩展特殊色**（共 70 色）。
- 色值为近似 HEX，颜色匹配采用 CIE-Lab 距离贴近人眼感受；实际购买请以厂商实物色卡为准。

## 🔧 自定义

- **新增图纸**：在 `src/app.js` 的 `LIBRARY` 数组里按字符画格式新增（`'.'` 表示空格，`legend` 定义每个字符对应的颜色）。
- **新增 / 修改颜色**：在 `src/app.js` 的 `MARD_PALETTE` 中增减 `{ code, hex }` 条目。
- **调整调色板组合**：修改 `PALETTES`（标准 / 全色卡 / 新手精选）。

## 📌 说明

- 本项目为纯前端静态站点，不收集任何数据；上传的图片仅在本地浏览器内处理。
- 色卡数据来源：[Zippland/perler-beads](https://github.com/Zippland/perler-beads)、[maxcleme/beadcolors](https://github.com/maxcleme/beadcolors) 的 MARD 色号映射。
