使用说明

拼豆工坊是一款在线拼豆图纸工具，可将图片转换为拼豆图纸、统计颜色用量，并支持在线编辑与现成图纸库。所有处理均在本地浏览器完成，无需联网。

一、软件简介

本工具面向拼豆爱好者，主要解决三个需求：① 把任意图片转成可照着拼的图纸；② 算清每种颜色需要多少颗豆；③ 对图纸进行个性化修改。无需安装，双击 index.html 即可在浏览器中使用。

二、快速开始

  1. 进入「制作图纸」页，点击或拖拽上传一张图片。
  2. 选一个快速档位（小 48 / 中 80 / 大 120 / 超清 160），或拖动「横向豆数」自定义精细度。
  3. 点击「生成拼豆图纸」，中间画布显示图纸，右侧显示各颜色用量。
  4. 如需修改，用左下角的编辑工具（画笔 / 橡皮 / 取色 / 填充）在画布上点击或拖拽。
  5. 满意后在右侧「导出 / 保存」导出 PNG 图纸或 CSV 用量清单。

三、功能说明

3.1 图片转图纸

参数作用
横向豆数图纸宽度（格子数）。可拖滑块（8–200）或在数字框直接输入（最高 400）。数值越大越清晰，但用豆更多、成品更大。
快速档位一键设定常用尺寸：小 48 / 中 80 / 大 120 / 超清 160。
保持原图比例勾选后高度按原图比例自动计算；取消可手动设定高度。
调色板采用 MARD 真实色卡，可选 MARD 标准 A–M（221 色） / MARD 全色卡（291 色） / 新手精选（72 色）。
对比度增强 / 色彩饱和度适当提高可让复杂图片更鲜明、避免量化后发灰发闷（默认已各加一点）。
锐化强化边缘细节，照片类图片边缘发虚时可勾选。
去除浅色背景把接近白色的背景留空不放豆，适合主体清晰、底色干净的图。
抖动 / 平滑「无」色块清晰；「扩散抖动」用混色让渐变更细腻。

提示：大图会先做分步降采样再匹配颜色，相比一次性缩放能保留更多细节，减少“糊成一团”。

3.2 用量统计

生成或编辑后，右侧实时显示总豆数、颜色数、图纸尺寸，并按用量从多到少列出每种颜色的色号与数量（占比条直观对比）。点击任意颜色一行，即可将其设为当前画笔颜色。

3.3 在线编辑

生成图纸后，左下角出现编辑工具。先在调色板中选择颜色，再选择工具在画布上操作：

工具说明
🖌️ 画笔点击或按住拖拽，给格子涂上当前颜色。
🧽 橡皮清除格子（变为空格，不放豆）。
💧 取色点击某格，把它的颜色设为当前画笔颜色。
🪣 填充把相连的同色区域一次性替换为当前颜色。

画布上方可切换网格线 / 色号 / 坐标尺，并进行缩放与「适应窗口」。勾选「色号」会在每格显示对应 MARD 色号（格子太小时会自动放大以便看清）。

3.4 图纸库

「图纸库」页内置 32 款现成图纸，分为经典 / 表情 / 动物 / 食物 / 植物 / 节日 / 游戏七类，点击顶部分类标签可筛选。点「载入并编辑」即可作为模板查看用量或二次创作；载入时颜色会自动吸附到最接近的 MARD 色号。

3.5 导出与保存

操作说明
导出图纸 PNG导出带网格线的高清图纸图片，便于打印对照。若已勾选「色号」，图片每格会标注 MARD 色号。
导出用量 CSV导出每种颜色的色号、色值、数量，可用 Excel 打开备料。
保存 / 载入项目把当前图纸保存为 JSON（并存入浏览器本地），日后可载入继续编辑。
清空画布清除全部格子，可用撤销恢复。

四、快捷键

Ctrl + Z 撤销，Ctrl + Y（或 Ctrl + Shift + Z）重做。

五、常见问题

Q：生成的图纸颜色偏差较大 / 糊成一团？
A：提高「横向豆数」（如 120–160）、开启「扩散抖动」，并适当增加对比度/饱和度；边缘发虚可再勾选「锐化」。建议使用对比度高、主体清晰的图片（卡通、像素图、Logo 效果最佳）。

Q：图纸太大不好拼？
A：降低「横向豆数」或选用「新手精选（72 色）」调色板减少颜色种类。

Q：勾选色号后看不清 / 想打印带色号的图纸？
A：勾选「色号」会自动放大画布显示；导出 PNG 时若保持勾选，图片每格也会带色号。

Q：保存的项目在哪里？
A：点击保存会下载一个 .json 文件，同时存入当前浏览器本地；下次用「载入已保存项目」选择该文件即可恢复。

🎨 内置色号采用 MARD 拼豆色卡，色号仅为字母 + 数字（如 A01、F03、ZG3），按色系字母分组：A–H、M 系为标准 221 色套装（A 黄橙系、B 绿系、C 蓝青系、D 蓝紫系、E 粉红系、F 红系、G 灰白黑系、H 紫系、M 灰系等），P/Q/R/T/Y/ZG 为扩展特殊色（合计 291 色）。色值为近似 HEX，颜色匹配采用 CIE-Lab 距离贴近人眼感受；实际购买请以厂商实物色卡为准。
